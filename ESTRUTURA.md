# 🧠 📌 INSTRUÇÃO PARA OS ALUNOS (pode colar)

## 🎯 Objetivo

Desenvolver uma solução digital (web e mobile) para resolver um problema **real**, social ou ambiental, documentando todo o processo de UX/UI em um repositório no GitHub.

---

## 📋 Entregáveis obrigatórios

Você deve incluir:

### 1. Problema

* Descrição clara do problema
* Contexto
* Quem é afetado
* Por que é relevante

---

### 2. Persona

* Nome fictício
* Idade
* Contexto
* Objetivo
* Dores
* Comportamento

---

### 3. Fluxo do Usuário (diagrama)

* Caminho completo do usuário
* Etapas principais
* Decisões

---

### 4. Wireframes

* Estrutura das telas
* Organização da informação
* Sem foco visual (baixo nível)

---

### 5. Protótipo

* Interface visual (UI aplicada)
* Navegação funcional
* Web + Mobile

---

### 6. Link do Figma

* Arquivo organizado
* Protótipo navegável

---
